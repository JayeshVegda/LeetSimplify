üì¶ Installation

From Source

  1. Clone the repository: 
    git clone https://github.com/JayeshVegda/LeetSimplify.git
cd LeetSimplify
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" (toggle in top right)
  4. Click "Load unpacked" and select the LeetSimplify directory

üöÄ Quick Start

  1. Click the extension icon to open settings
  2. Select your preferred provider (Gemini, ChatGPT, Claude, etc.)
  3. Enter your API key
  4. Choose simplification mode (No Hint / With Hint)
  5. Click "Save Settings"
  6. Navigate to a LeetCode problem and click "Simplify"

üîí Privacy

Cloud providers: Problem descriptions are sent to the selected API provider.

Local LLMs (Ollama): Complete privacy - all processing happens on your machine. No data leaves your computer.
